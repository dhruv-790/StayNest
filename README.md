🏡 StayNest – Airbnb-Inspired Full-Stack Web Application

🚀 Developed a scalable Airbnb-like platform enabling users to list, discover, and book accommodations with a complete end-to-end workflow, including secure authentication and online payments.

🔑 Key Contributions & Features
🔐 Implemented secure authentication & authorization using Passport.js with session-based login
🧩 Designed RESTful APIs and followed MVC architecture for a modular and maintainable backend
🏠 Built a dynamic listings system with full CRUD operations
☁️ Integrated Cloudinary for image upload, storage, and optimization
🔍 Developed search & filtering functionality to improve user discovery experience
📅 Created a booking management system with a personalized user dashboard
💳 Integrated Razorpay payment gateway for seamless transactions (test & live modes)
⭐ Added reviews & ratings system for user-generated feedback
🛠️ Designed an admin dashboard to monitor users, listings, bookings, and platform activity
📱 Developed a responsive UI using Bootstrap for cross-device compatibility
🛠️ Tech Stack
💻 Frontend: HTML, CSS, Bootstrap, EJS
⚙️ Backend: Node.js, Express.js
🗄️ Database: MongoDB (Mongoose)
🔐 Authentication: Passport.js
☁️ Media Storage: Cloudinary
💳 Payments: Razorpay
📈 Impact
🎯 Delivered a production-style full-stack application simulating real-world booking platforms
⚡ Demonstrated end-to-end workflow implementation, from authentication to payments
📊 Showcased scalable architecture and clean UI/UX design principles
